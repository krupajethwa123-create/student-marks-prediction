# student-marks-prediction
Description: This project predicts student performance based on study-related factors such as study hours and attendance. It uses Logistic Regression to classify outcomes and understand how input features affect performance.

Problem Statement: To predict whether a student will pass or perform well based on study hours and attendance.

Tools & Technologies
Python
Pandas
NumPy
Scikit-learn
Model Used
Logistic Regression

Steps Followed: 
Data loading and preprocessing
Exploratory Data Analysis (EDA)
Feature selection
Model training using Logistic Regression
Model evaluation using accuracy, precision, and recall

Outcome: The project helped in understanding classification problems and interpreting model predictions.
